                            p r o d u c t i o n s   
 ------------------------------------------------------------------------------
           we would like you to check these important projects :
  	

  rdosumb      utility, that gives you up to 192kb additional upper (umb) 
               memory without a need of protected mode managers such as qemm 
	       or emm386. useful for having 620kb+ base memory with himem.
	
  	       (current version of jan 97, archive name: rumb11ea.zip)  
  

  rdosplay     first generic music player for adlib compatible (opl2/opl3)
	       cards in pc world at all. module packing, graphical scopes and
	       over 16 fm-music formats (agd,hsc,rad,d00...) are supported
               together with easy_to_use file selector.

               (current version of jan 97, archive name: rplay017e.zip)


  remus        card1 -> card2 protected mode soundcard emulator. currently
               the most working emulation is the gus (gf1) emulation on sb, 
	       d/a, but the other could be useful for someone, too... 
               this project is still very ALPHA state
 
               (current version of jan 97, archive name: remus04e.zip) 



    last but not least, some demo related artwork has to be released soon.
